# Airbnb Performance Dashboard

An interactive Power BI dashboard analyzing Airbnb’s global performance, focusing on growth trends, city-level insights, user behavior, and trust metrics.

---

## 🎯 Project Overview

This project explores Airbnb’s journey through growth, regulation challenges, and the COVID-19 impact. It provides insights into market concentration, user engagement, and platform trust using data-driven storytelling.

---

## 🛠️ Tech Stack

- Power BI Desktop  
- Power Query  
- DAX (Data Analysis Expressions)  
- Data Modeling  

---

## 📂 Data Source

Airbnb Listings & Reviews Dataset from Maven Analytics

---

## 📌 Features / Highlights

### 🔹 Business Problem
Understanding Airbnb’s growth, user behavior, and trust across cities to improve platform performance and decision-making.

---

### 🔹 Goal of the Dashboard
To provide a comprehensive view of Airbnb’s performance by analyzing trends, city-level insights, and user engagement patterns.

---

### 🔹 Key Visuals

- **Overview Page**  
  Shows Airbnb lifecycle (Growth → Decline → COVID impact) and listing trends  

- **Ratings Page**  
  Pareto analysis, pricing comparison, and ratings insights  

- **Review Page**  
  Reviewer segmentation, trust analysis, and seasonality trends  

---

### 🔹 Business Impact & Insights

- Listings peaked in 2015 and declined during COVID  
- A few cities dominate the market (~50% contribution)  
- Most users contribute only one review → low engagement  
- High-frequency reviewers are minimal  
- Trust varies across cities  
- Seasonal patterns reflect travel demand  

---
